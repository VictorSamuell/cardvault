# CardVault

**URL do site:** [CardVault](https://cardvault-liart.vercel.app/)

## Tecnologias Utilizadas
- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express 
- **Deploy:** Vercel + Render
- **Banco de Dados:** MongoDB

## Como Usar
1. Acesse o site pelo link: [CardVault](https://cardvault-liart.vercel.app/)
2. Crie conta ou faça login
3. Crie coleções, explore coleções ou veja sets/cartas
